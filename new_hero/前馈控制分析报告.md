# 云台PITCH和YAW前馈控制分析报告

## 1. 当前控制系统分析

### 1.1 控制结构
- **PITCH轴**：角度环PID直接输出扭矩（单环控制）
  - 角度环PID输出直接作为电机扭矩
  - 使用IMU的pitch角度作为反馈
  - 没有速度环，也没有前馈控制

- **YAW轴**：角度环+速度环串级控制
  - 角度环PID输出速度设定值
  - 速度环PID输出扭矩
  - 在小陀螺模式和跟随底盘模式中有底盘角速度前馈

### 1.2 当前前馈控制实现
1. **小陀螺模式**：添加底盘角速度前馈补偿（feedforward_gain = 1.0f）
2. **跟随底盘模式**：添加底盘角速度前馈补偿（feedforward_gain = 0.2f）
3. **正常模式**：没有前馈控制
4. **PITCH轴**：完全没有前馈控制

## 2. 前馈控制设计原则

### 2.1 前馈控制的作用
1. **提高响应速度**：提前补偿已知的干扰
2. **减小跟踪误差**：补偿系统动态特性
3. **提高抗干扰能力**：抵消外部干扰

### 2.2 需要的前馈类型
1. **目标速度前馈**：基于目标位置的变化率
2. **目标加速度前馈**：基于目标位置的变化加速度
3. **干扰前馈**：补偿已知的外部干扰（如底盘旋转）

## 3. 前馈控制设计方案

### 3.1 PITCH轴前馈控制
```c
// 目标速度前馈计算
float pitch_target_velocity = (pitch_target_pos - last_pitch_target_pos) / dt;

// 前馈扭矩计算
float pitch_feedforward_torque = 
    pitch_target_velocity * Kp_ff +  // 速度前馈
    pitch_target_acceleration * Ka_ff; // 加速度前馈

// 总扭矩输出
pitch_motor_torque = PID_output + pitch_feedforward_torque;
```

### 3.2 YAW轴前馈控制
```c
// 目标速度前馈（角度环）
float yaw_target_velocity = (yaw_target_pos - last_yaw_target_pos) / dt;

// 角度环前馈
float angle_ff = yaw_target_velocity * Kp_angle_ff;

// 速度环前馈（基于角度环输出变化率）
float velocity_ff = (angle_ff - last_angle_ff) / dt * Kv_ff;

// 干扰前馈（底盘角速度）
float disturbance_ff = chassis_yaw_rate * Kd_ff;

// 总速度设定值
yaw_velocity_setpoint = PID_angle_output + angle_ff + disturbance_ff;

// 速度环前馈
float velocity_loop_ff = velocity_ff;
```

### 3.3 不同模式的前馈策略
1. **正常模式**：目标速度前馈 + 目标加速度前馈
2. **小陀螺模式**：目标速度前馈 + 底盘角速度前馈
3. **跟随底盘模式**：目标速度前馈 + 底盘角速度前馈
4. **自瞄模式**：视觉目标速度前馈

## 4. 具体实现方案

### 4.1 数据结构扩展
```c
typedef struct {
    // 前馈控制参数
    float Kp_ff;        // 位置前馈增益
    float Kv_ff;        // 速度前馈增益
    float Ka_ff;        // 加速度前馈增益
    float Kd_ff;        // 干扰前馈增益
    
    // 状态记录
    float last_target_pos;
    float last_target_vel;
    float last_angle_output;
    
    // 滤波器
    first_order_filter_type_t target_vel_filter;
    first_order_filter_type_t target_acc_filter;
} feedforward_control_t;
```

### 4.2 前馈计算函数
```c
float calculate_feedforward_torque(feedforward_control_t *ff, 
                                   float target_pos, 
                                   float dt) {
    // 计算目标速度（带滤波）
    float target_vel = (target_pos - ff->last_target_pos) / dt;
    first_order_filter_cali(&ff->target_vel_filter, target_vel);
    target_vel = ff->target_vel_filter.out;
    
    // 计算目标加速度
    float target_acc = (target_vel - ff->last_target_vel) / dt;
    first_order_filter_cali(&ff->target_acc_filter, target_acc);
    target_acc = ff->target_acc_filter.out;
    
    // 更新状态
    ff->last_target_pos = target_pos;
    ff->last_target_vel = target_vel;
    
    // 计算前馈扭矩
    return target_vel * ff->Kv_ff + target_acc * ff->Ka_ff;
}
```

## 5. 参数调优建议

### 5.1 初始参数
```
// PITCH轴前馈参数
PITCH_KV_FF = 0.1f    // 速度前馈增益
PITCH_KA_FF = 0.01f   // 加速度前馈增益

// YAW轴前馈参数
YAW_KP_ANGLE_FF = 0.5f  // 角度环速度前馈增益
YAW_KV_FF = 0.05f       // 速度环前馈增益
YAW_KD_FF = 1.0f        // 干扰前馈增益（小陀螺模式）
```

### 5.2 调优步骤
1. 先调速度前馈增益（Kv_ff）
2. 再调加速度前馈增益（Ka_ff）
3. 最后调干扰前馈增益（Kd_ff）
4. 根据实际响应调整滤波器参数

## 6. 预期效果
