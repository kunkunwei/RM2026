# Wheel-Leg Infantry Robot Control Firmware (轮足步兵机器人底盘控制)

## 项目简介
本项目为五连杆并联腿轮足机器人（RoboMaster 步兵机器人）底盘控制固件，主控为 **STM32F407IGH6**，基于 **FreeRTOS** 与 **HAL**。
核心控制采用离线 **LQR** 增益表插值，兼顾稳定性与实时性。

## 硬件平台
- **MCU**: STM32F407IGH6 (ARM Cortex-M4, 168MHz)
- **构型**: 五连杆并联腿 + 轮足 (Diamond/V-shape linkage)
- **电机**:
  - 关节电机 x 4 (左右各2)
  - 轮毂电机 x 2 (足端)

## 控制与通信
### 1) 底盘输入控制方式切换与硬件支持
- **输入来源**: 遥控器、上位机/裁判系统、云台控制板。
- **切换逻辑**: 由底盘行为/状态机统一管理，在 `chassis_behaviour.c` 中完成控制来源仲裁与模式切换。
- **硬件支持**:
  - **CAN**: 底盘与云台/裁判系统/电机驱动通信主通道。
  - **UART**: 调参与上位机调试，亦可用于扩展输入。
  - **SBUS/PPM**: 遥控器输入（若已接入，对应 BSP/UART 口）。

### 2) 云台与底盘的通信控制
- **通信介质**: 通过 **CAN** 完成云台-底盘指令交互。
- **典型内容**: 速度设定、姿态/角度指令、状态回传、模式切换触发。
- **位置**: 参考 `Application/Tasks/` 中底盘任务与行为文件的通信处理逻辑。

### 3) 底盘状态机与状态机控制
- **状态机位置**: `chassis_behaviour.c`。
- **常见状态**: 无力/起立/跟随云台/自旋等（具体以代码枚举为准）。
- **作用**: 统一管控输入来源、速度/腿长/角度目标设定以及保护策略。

### 4) 小陀螺前进功能说明
- 小陀螺前进功能 **尚未完成**，需要进一步研究**差速控制相关论文**后完善。

### 5) 质量变化与 LQR 参数
- 当步兵**整体质量有大幅变化**时，需要修改 `LQR_Balance_Turn` 函数中的 **coefficient 系数矩阵**。

### 6) 电机 ID 与零点基准
- **ID 顺序**（从机体后方俯视，顺时针）：右侧靠近电池为电机1，右后方为电机2，左后方为电机3，左侧靠近电池为电机4。
- **零点基准**：以五连杆运动学解算的平面为基准。
- **代码补偿**：在 `Bsp/Src/bsp_can.c` 的 `CAN_Process_Data` 内已做电机零点与顺序补偿。

### 7) 双路 CAN 的必要性与负载调节
- **使用两路 CAN 控制 2/4 个电机的原因**：
  - 单路 CAN 在跳跃时总线负载可达 100%，导致丢帧，C 板难以获取总线控制权。
  - 两个关节电机控制会出现相位延迟，跳跃更容易失败。
- **负载监测建议**：使用达妙 USB 转 CAN 查看总线负载。
- **阻抗调节**：通过分电板上的 **120 欧姆电阻开关**调整总线阻抗以稳定负载。
- **负载建议区间**：稳定在 **70% 左右**较稳；若负载在 **30%~70%** 间大幅跳变或达到 **90%+**，需调整电阻或降低挂载电机数量。

## 项目再生成注意事项
如果需要重新生成项目：
1. **先在本地构建项目**，确保当前工程可正常编译。
2. 再使用 **STM32CubeMX** 重新生成。
3. 生成后需将 `CMakeLists.txt` **回溯/恢复到手写版本**，以免被覆盖。

## 并联腿 -> 串联腿移植要点
若需将本并联腿机器人移植到串联腿机器人，请按以下流程修改：
- **导入新的运动学正/逆解算**。
- **导入新的 LQR 多项式拟合函数**。
- **重新定义电机零点**。
- **修改角度预测控制器参数与角度力矩补偿**。
- **可移除打滑检测器**（原用于限制足端疯狂打转）。
- **跳跃函数**: 仅做部分修改（按腿杆推力分阶段修正 + 机械限位思路）；也可**直接去掉跳跃函数**，用“撞上台阶”的方式实现地形跨越。
- **修改状态机并增加自救状态机**。

## 代码结构速览
- `Application/`: 应用层
  - `Tasks/`: FreeRTOS 任务
    - `chassis_task.c`: 底盘控制主循环
    - `chassis_behaviour.c`: 底盘行为状态机
- `Bsp/`: 板级支持包 (CAN/SPI/UART/PWM 等)
- `Components/`: 组件层 (算法/控制器/设备)
- `Matlab_lib/`: MATLAB 自动生成代码 (LQR/运动学)

## 调试说明
- **调试接口**: J-Link / ST-Link (SWD)
